# Sistema de Gestion Tecnica Profesional

## Descripcion

Sistema orientado a la gestion tecnica profesional para empresas de mantenimiento y soporte tecnico.

El sistema permite:
- Registrar clientes
- Gestionar ordenes tecnicas
- Asignar tecnicos
- Generar reportes
- Gestionar mantenimientos

## Actores

- Administrador
- Tecnico
- Cliente

## Requerimientos Funcionales

- Inicio de sesion
- Registro de clientes
- Registro de ordenes tecnicas
- Asignacion de tecnicos
- Generacion de reportes

## Requerimientos No Funcionales

- Escalabilidad
- Seguridad
- Mantenibilidad
- Modularidad

## Patrones de Diseño

- Singleton
- Facade
- Observer
- Strategy
